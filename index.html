Vinheria Agnello

Sistema de avaliacao de estoque e tendencia de vinhos

Sobre o projeto

Esta pagina apresenta um sistema simples feito em JavaScript para cadastrar vinhos, verificar estoque baixo, classificar a safra e exibir um resumo final pelo navegador.

O que o sistema faz

  • Cadastra varios vinhos usando prompt.
  • Valida entradas de texto e numero.
  • Identifica estoque baixo quando a quantidade e menor que 5.
  • Classifica o vinho como jovem, amadurecido ou antigo.
  • Mostra os dados no console e o resumo final em alert.

Como testar

Ao abrir esta pagina no navegador, responda aos prompts. Para ver os detalhes cadastrados, abra o console com F12 e acesse a aba Console.
